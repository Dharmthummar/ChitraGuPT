How it works

Three simple steps to automated data entry.

1. Upload Document

Drag and drop any image, invoice, or screenshot into the secure vault.

2. AI Processing

Gemini 3.1 Flash Lite analyzes the context and extracts structured data intelligently.

3. Excel Export

Data is automatically appended to your target spreadsheet. No manual entry.

import { GeminiExtractor } from '@ai/vision'; const invoice = await GeminiExtractor.parse('receipt_01.png'); console.log(invoice.data); // Output: { "vendor": "OpenAI", "total": 120.50, "date": "2026-04-29" }
